 ##github 업로드 다운로드 가이드


1.    빠른 가이드

      1.  다운로드

      ```shell
      git init
      git remote add origin [URL]
      git fetch 
      ```

      2.  업로드

      ```shell
      git add *
      git config --global user.name "유저명" #한번만 설정하면 됨
      git config --global user.email "이메일" #한번만 설정하면 됨
      git commit -m "comment"
      git push origin master
      ```
